I built a Streamlit web app that automates Energybae's solar proposal workflow — the user uploads an electricity bill image, and the app uses Groq's Llama 4 Scout Vision AI to extract consumer details, 12 months of consumption history, and billing data automatically. The extracted data is injected into the provided Solar Load Calculator Excel template (preserving all formulas), and the user downloads a ready-to-use proposal Excel in one click. I used Python with Streamlit for the UI, Groq API for AI-powered bill reading, and openpyxl for formula-safe Excel manipulation. If I had more time, I would add PDF support, batch processing for multiple bills, deploy it as a hosted web app, and support electricity bills from other states beyond Maharashtra.
